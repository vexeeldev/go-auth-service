# backend

Simple Go REST API - Framework: chi | Database: postgres

## Quick Start

1. Edit `.env` file
2. Run: `make run`

## API Endpoints

- `GET /` - Home
- `GET /health` - Health
- `GET /api/users` - Get all users
- `POST /api/users` - Create user
- `GET /api/users/{id}` - Get user
- `DELETE /api/users/{id}` - Delete user
